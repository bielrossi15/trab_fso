## Compressor e Compactador .bz2.tar
### Como rodar
1. Compile o arquivo trabfinal.c
 ```
 gcc trabfinal.c -lm -O2 -static -lpthread -lbz2
 ```
 
2. Execute o programa
```
./a.out <arquivo de entrada> <arquivo de saída>
```
